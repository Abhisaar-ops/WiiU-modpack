This page is for Loadiine compatibility list.

Loadiine is a WiiU backup loader for SD Card, working on WiiU 5.3.2 to 5.5.1, and it's region free !

XML[edit]

Loadiine 4.0 and Loadiine GX2 use the game's cos.xml and app.xml to get the correct place to store the game in memory. The xml files can be extracted from a WUD (WiiU ISO) or from eShop updates. If you don't have the xml files in your wiiu/games/<game name>/code/ folder, loadiine will use commonly used values, but not all games will work with these default values.

If you don't have the xml file, here are some methods to get it:

  • Extract it from WUD (You need the disc key and DiscU, vgmtoolbox or Uwizard)
  • Extract it from eshop full game (you need the NUS key and Cdecrypt)
  • Extract it from eshop update (no NUS key needed)
  • Use JNUSTool to get the xml from eshop full game or eshop update. (you need the NUS key only if there is no game update)
  • Extract it from memory by running the original application with loadiine_xml_dumper
  • Copy the xml from a different game, but don't forget to edit the Title_ID. Start testing with Super Smash Bros. xml files.
  • Ask other users if they have successfully created/gotten their own, and to please share it with you
  • Create your own (try random values by looking at other game's xml common values)
  • Just hope for the best and try your luck with the default common values.

You can find some xml compilations here:

  • update_xml_compilation_19_11_15_eur-rar
  • update_xml_compilation_20_11_15_usa-rar
  • Virtual console XML files (needed for NDS and N64).

Posting Guidelines[edit]

Name - Add the name obviously and if it’s a game that has several releases like 1.0, 1.1 and 1.2 add it as well or the tmd version (v0 v64). You can use comment column for version info too.

Region - Enter "USA", "EUR", or "JAP". Leave empty if all region have the same compatibility. Add the region only for a different compatibility or if a different comment is needed.

Note : "USA/EUR" and "USA/EUR/JAP" are additional available entries in case of identical releases in multiple regions.

RPX - Enter the size of the RPX file from the initial version (game disc or Eshop v0), without applying the update. Size is in Bytes.

RPL - enter the size of each RPL file from the initial version (game disc or Eshop v0), without applying the update. Size is in Bytes.

Note: If a game has multiple RPL files, please specify all the file's sizes, not the addition of all. Example : not 360KB but 339KB + 21KB (or in bytes if possible)

Size - Enter game size (/content/ folder's size)

Version - Higher tested Loadiine version. Add in the comment column the last working version if a new Loadiine release breaks compatibility.

Disc (SSBU) - Whether the game works with Super Smash Bros. for WiiU disc (if you are using eShop version, add a note if SSBU eshop has a different compatibility than the disc mode)

MiiMaker - Whether the game works with MiiMaker mode

Notes - Use this to describe playability, bugs and glitches and to give additional settings if needed.

Compatibility Legend Class (?)
Games that work 100% or have few/very minor issues, like stuttering audio, or other things that don't affect gameplay. ct-works
Games that mostly work but with more serious but playable bugs, or random lockups (random doesn't mean lock up every time at the same spot in the game). ct-issues
Games that do not load and have major problems and/or are unplayable, or have consistent lock ups. ct-broken
Unknown status. Game has not been tested, or the test results were unclear. ct-noresult
